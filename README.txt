Kinematique - Handshake with Yahboom Dofbot Arm

How to run:
Assuming you are connected locally to the robot, open a terminal.
Navigate to this repository ($cd ~/kinematique\ FA22/).
Start Jupyter Notebook ($jupyter notebook).
If it didn't open the browser for you, go to localhost:8889.
Select a notebook (e.g. Driver) to open.
Select Cells>Run All.
